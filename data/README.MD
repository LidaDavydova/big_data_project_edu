# Columns chosen for the analysis

1. `nr_passag_pagos`: Number of passengers who occupy seats.
2. `nr_passag_gratis`: Number of passengers who occupy seats marketed to the public, but do not generate revenue.
3. `ds_servico_tipo_linha`: Description of the service referring to the type of line (Passenger/Cargo).
4. `kg_carga_paga`: Cargo paid. It is the total amount, expressed in kilograms, of all goods that have been transported on the aircraft, except mail and luggage, and have generated direct or indirect revenue for the airline. 
5. `id_aerodromo_origem`: Identifier code of the airfield of origin in the system.
6. `nm_pais_origem`: Name of the country of origin.
7. `id_aerodromo_destino`: Identification code of the destination aerodrome in the system.
8. `nm_pais_destino`: Name of the country of destination.
9. `ds_grupo_di`: Description of the DI group or Flight Type. Refers to the grouping of DI classifications into types of operations:
- Regular
- Non-regular
- Unproductive
10. `ds_natureza_tipo_linha`: Description of the nature referring to the type of line (International/Domestic).
11. `ds_tipo_empresa`: Description of the type of company. It refers to the description of the type of company in relation to the service performed. It may contain the following descriptions:
- Regular air transport
- Non-regular air transport
- Regular domestic
- Regular foreigner
- Non-regular foreigner
- Public air services
- Regular systematic airway connection
- Air taxi
- Aerial
- Aeropublicity
- Aerophotography
- Air inspection
12. `nr_ano_partida_real`: Number of the actual departure year.
13. `nr_mes_partida_real`: Number of the real departure month.
